# TBox 

## Prerequisites 

- nix 
- direnv

To start workign on repository enable the direnv to use the flake as dev environment.

```sh
direnv allow
```

## Running Code

```sh
bun ./src/server.ts
```

## Operations

**Building**

To build the package:

```sh
bun build
```

**Testing**

To test the package:

```sh
bun test
```
